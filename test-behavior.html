🧪 Test de Comportamiento - Quiz App

📋 Tests Disponibles

Este test verifica que la funcionalidad de modificación de respuestas funcione correctamente según la configuración.

🎯 Test 1: Modo Normal (Respuestas Inmediatas)

Comportamiento esperado:

  • ✅ Las respuestas se muestran inmediatamente
  • ✅ No se pueden cambiar respuestas una vez seleccionadas
  • ✅ La puntuación se muestra en tiempo real
  • ✅ Las opciones se deshabilitan después de seleccionar

🔒 Test 2: Modo Oculto (Respuestas al Final)

Comportamiento esperado:

  • ✅ Las respuestas NO se muestran inmediatamente
  • ✅ Se pueden cambiar respuestas en cualquier momento
  • ✅ Solo se muestra progreso, no puntuación
  • ✅ Las opciones permanecen activas

🔄 Test 3: Prueba Manual Completa

Prueba interactiva:

  • 📝 Carga el test-final.json en ambos modos
  • 🎮 Interactúa manualmente con la aplicación
  • ✔️ Verifica el comportamiento esperado
